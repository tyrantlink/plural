Privacy Policy for /plu/ral

I am not a web designer

1. Data Collection

/plu/ral collects data that users explicitly provide using interactions with the bot.

When a message is proxied, the following data is stored for one day:
- Original Message ID
- Proxied Message ID
- Channel ID
- User ID

When using the userproxy proxy command with the "queue_for_reply" option, the following data is stored for five minutes:
- Channel ID
- Queued Message Content
- Queued Attachments

When a message is proxied in a server with the "logclean" option enabled, the following data is stored for one minute:
- Original Message ID
- User ID
- Channel ID
- Author Username

2. Temporary Data Collection

/plu/ral collects and stores all data recieved from the Discord API for caching purposes. This data is stored temporarily and is automatically deleted one hour after it is received.
Message content and attachments are stripped from the data before it is stored.

3. Use of Data

The data collected is used strictly for bot functionality, such as proxying messages.

4. Open Source

Since /plu/ral is open source, the source code is freely available for review under the AGPL license. Users can see exactly how the bot operates and how data is handled. The source code is available on GitHub.

5. User Control and Data Deletion

Users have full control over their data and can delete all stored data using the /delete_all_data command.

6. Data Sharing

No data is shared with third parties, except as required by law.

7. Data Storage

Data is stored securely and is accessible only to the developers of /plu/ral.

8. Your Rights

You have the right to request the deletion of all your data by using the /delete_all_data command.
